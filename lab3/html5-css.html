HTML5 e CSS

HTML 5

O HTML5 é a nova versão do HTML4 e um dos seus principais objetivos é facilitar a manipulação dos elementos.

Elementos Uso Explicação
header ✔ indica cabeçalho (normalmente com logotipos).
nav ✔ indica menu de navegação com links para as páginas (só deve haver um por página)
main ✔ indica conteúdo principal e único, que não se repete.
article ❌ indica porção do documento que pode ser vista de forma independente
section ✔ especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|etc...
aside ❌ indica anotações, publicidade ou outros assuntos desconexos.
footer ❌ indica rodapé da secção (com copyright).

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

Tipo de Seletor Uso Explicação
Seletor Universal ✔ O seletor universal (*) aplica o estilo a todos os elementos da página.
Seletor de Tipo ✔ O seletor de tipo usa o nome do marcador a estilizar. A regra será aplicada a todos os elementos que possuam esse marcador
Seletor de Classe ✔ O seletor de classe associa um nome a um estilo. No seletor de classe, o nome (case-sensitive) é antecedido de um "."
Selector de id ✔ O atributo id permite identificar univocamente um elemento do HTML (muito útil no JavaScript). Neste o seletor, o identificador (case-sensitive) é antecedido de #.
Selectores de Atributo ✔ Seletores que se aplicam a atributos de elementos
Selectores de Pseudo-classe ✔ Seletor associado ao estado de um elemento
Selectores de Pseudo-Elementos ✔ Regras que se aplicam a partes de elementos.
